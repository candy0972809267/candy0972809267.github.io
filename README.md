# MOFF 美食模擬點餐網頁
希望忙碌的人們也能吃到想吃的美食，因此有了MOFF美食模擬點餐網頁的誕生! <br>
為您整合桃園市的優質餐廳，以平價、中價、高價來分類，透過簡潔的圖片與店家連結，讓人們可以快速瀏覽並選擇今天想吃什麼。<br>
結合外送服務，讓您可以直接在線上訂餐，再忙碌都能品嘗到美味的食物！透過我們的網頁，MOFF將為您實現飯來張口的外送服務!
(此網頁無後台，僅供前台模擬)
